# Echo Vision

An AI-powered image captioning system using vision and language models. Generate descriptive captions for images using different approaches.

## Files
- `echo_vision_hf_only.ipynb` - Image captioning using HuggingFace Transformers
- `echo_vision_google_gen_ai_and_hf.ipynb` - Image captioning using Google Gemini and HuggingFace
- `images/` - Sample images for captioning
