# ClinRAG: Clinical Knowledge Assistant

A RAG-based medical question answering system using Alzheimer's disease content. Ask questions and get accurate, context-grounded answers from the medical knowledge base.

## Files
- `ClinRAG.ipynb` - Main Q&A system
- `knowledge_indexing.ipynb` - Vector store creation
- `data/` - Medical text documents
- `vector_store/` - ChromaDB vector database
